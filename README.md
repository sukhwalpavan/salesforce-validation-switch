# Salesforce Validation Rule Switch App

This project is a web application that connects with a Salesforce Developer Org using OAuth 2.0 and Salesforce Tooling API.

## Features

- Login with Salesforce
- Fetch validation rules from Salesforce
- Show validation rule active/inactive status
- Activate and deactivate validation rules
- Update changes directly in Salesforce

## Tech Stack

### Frontend
- React JS
- Axios
- Vite

### Backend
- Java
- Spring Boot
- WebClient
- Salesforce OAuth 2.0
- Salesforce Tooling API
